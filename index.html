Problem statement

How to perform a CPU intensive task on the client side using the single-thread language?

Web Worker API

                    
                        const worker = new Worker();
                        worker.onmessage = function(e) { /* ... */ };
                        worker.postMessage(message, [transfer]);
                        worker.terminate();

Restrictions:

  • Web Workers don’t have access to the window
  • Web Workers don’t have access to the DOM
  • Communication between Web Workers and main thread is only performed by messages.
